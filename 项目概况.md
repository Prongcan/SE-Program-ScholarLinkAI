# ScholarLink AI - 项目概况

## 🎯 项目简介

**ScholarLink AI** 是一个智能学术论文推荐与解读平台，通过 AI 技术帮助研究人员快速发现和理解最新的学术论文。

### 核心功能

1. **📄 论文自动抓取** - 从 arXiv 自动抓取最新的 CS 类学术论文
2. **👤 用户兴趣管理** - 用户可设置研究兴趣，系统基于兴趣推荐论文
3. **🤖 AI 论文解读** - 使用 OpenAI GPT 自动生成论文的中文解读博客
4. **💡 智能推荐** - 基于用户兴趣和论文内容的智能推荐系统
5. **🎨 现代化界面** - 仿 PaperIgnition 风格的简洁美观前端

## 💾 数据库设计

### 表结构

#### 1. papers (论文表)
```sql
paper_id     INT PRIMARY KEY AUTO_INCREMENT
title        VARCHAR(1000)  -- 论文标题
author       VARCHAR(1000)  -- 作者列表
abstract     TEXT           -- 摘要
pdf_url      VARCHAR(512)   -- PDF 链接
```

#### 2. users (用户表)
```sql
user_id      INT PRIMARY KEY AUTO_INCREMENT
username     VARCHAR(100) UNIQUE  -- 用户名
password     VARCHAR(255)         -- 密码(哈希)
interest     VARCHAR(255)         -- 研究兴趣
```

#### 3. recommendations (推荐表)
```sql
id           INT PRIMARY KEY AUTO_INCREMENT
user_id      INT              -- 用户 ID (外键)
paper_id     INT              -- 论文 ID (外键)
blog         TEXT             -- AI 生成的博客内容
created_at   TIMESTAMP        -- 创建时间
```

## 🔌 核心 API 接口

### Papers API (`/api/papers/`)

| 方法 | 端点 | 功能 | 说明 |
|-----|------|------|------|
| POST | `/fetch` | 抓取论文 | 从 arXiv 抓取并保存论文 |
| GET | `/list` | 论文列表 | 分页查询论文 |
| GET | `/<paper_id>` | 论文详情 | 获取单篇论文信息 |

**示例**：
```bash
# 抓取 10 篇论文
curl -X POST http://localhost:3001/api/papers/fetch \
  -H "Content-Type: application/json" \
  -d '{"max_results": 10}'

# 查看论文列表
curl http://localhost:3001/api/papers/list?page=1&page_size=20
```

### Users API (`/api/users/`)

| 方法 | 端点 | 功能 | 说明 |
|-----|------|------|------|
| POST | `/register` | 用户注册 | 创建新用户 |
| GET | `/<user_id>` | 用户信息 | 获取用户详情 |
| PUT | `/<user_id>/interest` | 更新兴趣 | 修改用户研究兴趣 |
| GET | `/<user_id>/interest` | 获取兴趣 | 查询用户兴趣 |
| GET | `/list` | 用户列表 | 分页查询用户 |

**示例**：
```bash
# 注册用户
curl -X POST http://localhost:3001/api/users/register \
  -H "Content-Type: application/json" \
  -d '{
    "username": "researcher",
    "password": "password123",
    "interest": "Machine Learning, NLP"
  }'

# 更新兴趣
curl -X PUT http://localhost:3001/api/users/1/interest \
  -H "Content-Type: application/json" \
  -d '{"interest": "Deep Learning, Computer Vision"}'
```

- **Framework**: Flask 2.3.3

## 🚀 快速启动

### 1. 环境准备


```yaml

  port: 3306
```

## 📊 工作流程

### 典型使用场景

```
1. 用户注册并设置研究兴趣
   └→ POST /api/users/register
   
2. 系统抓取最新论文
   └→ POST /api/papers/fetch
   
3. 用户浏览论文列表
   └→ GET /api/papers/list
   
4. 查看论文详情
   └→ GET /api/papers/<id>
   
5. AI 生成论文解读博客
   └→ Generate_blogs.py (调用 OpenAI API)
   
6. 保存推荐和博客
   └→ INSERT INTO recommendations
   
7. 基于兴趣推荐论文
   └→ embedding.py (向量相似度匹配)
```
